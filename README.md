# Textové RPG napsané v TypeScriptu na kroužku Tvorba webovek, Intellectus

Jak hrát:
 1. `git clone https://github.com/m93a/web-textbased-rpg`
 2. otevřete v prohlížeči `index.html`

Jak vyvíjet
 1. `git clone https://github.com/m93a/web-textbased-rpg`
 2. `yarn`
 3. udělejte změny
 4. `yarn build`
 5. otevřete v prohlížeči `index.html`
